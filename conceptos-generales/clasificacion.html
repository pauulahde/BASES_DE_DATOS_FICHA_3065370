Clasificación de bases de datos

La clasificación de bases de datos puede hacerse según diferentes criterios:

  1. Según su estructura Relacionales (SQL) → MySQL, PostgreSQL, Oracle. No relacionales (NoSQL) → MongoDB, Cassandra.
  2. Segun su ubicacion Locales → Instaladas en un solo equipo (Ejemplo: Microsoft Access). Distribuidas → Almacenadas en varios servidores (Ejemplo: Amazon DynamoDB).
  3. Según su finalidad Transaccionales → Usadas en operaciones comerciales diarias, como sistemas bancarios. Analíticas → Para análisis de grandes volúmenes de datos, como los almacenes de datos empresariales.
  4. Según el tipo de acceso Centralizadas → Toda la información en un solo lugar (Ejemplo: una base de datos de empleados en una empresa). Descentralizadas → Información dividida en diferentes ubicaciones para mayor disponibilidad y seguridad.
